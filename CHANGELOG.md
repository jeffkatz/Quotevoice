# Changelog

All notable changes to Quotevoice will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-10

### Added
- **Invoice Generator** - Create professional invoices with automatic numbering
- **Quotation Generator** - Generate quotes with QT-prefix numbering
- **Client Management** - Add, edit, and search clients
- **Dashboard** - View revenue, overdue invoices, and drafts at a glance
- **Settings Panel** - Configure company details, banking info, and tax rates
- **Logo Upload** - Brand documents with your company logo
- **PDF Export** - Download publication-ready PDF documents
- **Print Support** - Direct printing from preview
- **Offline Storage** - All data saved locally in JSON format
- **South African Defaults** - ZAR currency, 15% VAT pre-configured

### Technical
- Built with Electron 33.x and React 18.x
- TypeScript for type safety
- Tailwind CSS for styling
- Vite for fast development builds
- electron-builder for Windows packaging
